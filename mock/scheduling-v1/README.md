# Mock Payloads (scheduling-v1)

These files are sample request/response payloads aligned with `openapi/scheduling-v1.yaml`.

## Naming
- `*_200.json` / `*_202.json`: success response samples
- `*_request.json`: sample request payloads (for POST endpoints)
- `common_error_400.json`: common validation error payload

## Scope
- ERP inbound list APIs
- MES inbound list APIs
- ERP writeback APIs
- Internal event APIs
