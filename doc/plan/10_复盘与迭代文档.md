# 复盘与迭代文档（模板）
> 文档属性：模板文档（用于复盘记录），不直接定义实施口径。
> 当前实施口径以 `doc/00_现行口径基线_2026-03-21.md` 为准。
> 本文中的“建议”均为模板占位词，不表示当前版本存在待拍板事项。

## 1. 复盘目标
- 评估本阶段上线效果是否达成目标。
- 识别问题根因并转化为可执行迭代项。

## 2. 指标复盘（建议）
- 计划达成率
- 准交率
- 延期订单数
- 插单影响订单数
- 重排次数
- 回写成功率
- 人工干预次数

## 3. 问题归类
1. 数据问题：字段缺失、口径不一致、关联失败。  
2. 规则问题：优先级不符合业务预期、锁单策略不合理。  
3. 流程问题：审批慢、口头信息未结构化。  
4. 系统问题：性能、稳定性、可用性。

- 优先降低人工口头干预比例。
