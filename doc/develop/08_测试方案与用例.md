# 测试方案与用例（开发版）

## 1. 测试范围
- 功能：待排池、规则引擎、自动排产、调序、版本发布、报工反馈、重排。
- 接口：ERP/MES入站与ERP回写。
- 数据：字段完整性、口径一致性、跨表关联。

## 2. 测试分层
1. 单元测试：规则评分、依赖关系、时间分配。  
2. 集成测试：接口鉴权、分页、增量、重试、幂等。  
3. 端到端测试：完整业务闭环（含班次与重排）。  
4. 回归测试：规则变更与发布后稳定性。  

## 3. 关键用例清单
1. 正常订单自动排产并发布成功。  
2. 加急订单插单后影响分析正确。  
3. 锁单后自动重排不影响该订单。  
4. 串行工序顺序正确，禁止反向穿越。  
5. 并行工序在并发上限内正确排布。  
6. MES报工后进度更新正确。  
7. 延期阈值命中后生成重排候选版本。  
8. ERP回写失败后可重试补偿。  
9. 系统排产结果与历史Excel样例抽样比对（订单顺序、完工日期、异常差异）。  
10. 外围节点（包装/灭菌/入库）状态流转正确。  
11. 计划订单投放后可生成并接入生产订单；未投放不进入待排池。  
12. 排产对象校验：生产用料清单不直接参与排产计算。  
13. 触发同步与夜间补偿同步均可稳定执行。  
14. 单次排产计算性能满足 P95 <= 5 分钟。  
15. 班次口径校验：`1班次=12小时`，`1天1班/2班` 产能计算正确。  
16. 人机料约束校验：资源任一不足时任务不可开工并产生明确原因。  
17. 多技能人员分配校验：同一员工同一时段不被重复占用。  
18. 设备多工艺冲突校验：同一设备同一时段只允许一个工艺任务。  
19. 缺料占位校验（P1增强，P0非阻断）：缺料任务可排队占位但不可执行。  
20. WIP批次校验：工序完工可生成唯一批次并可追踪到下道工序。  
21. 重排分级校验：轻微偏差仅局部重排，重大异常触发全局重排。  
22. 冻结窗校验：未来24小时自动重排不改动冻结窗内任务。  
23. What-if仿真校验（P2增强，P0非阻断）：仿真结果与正式排产可区分，且不污染生产数据。  

## 4. 数据质量用例
- 必填字段为空拦截。
- 主键重复拦截。
- 日期格式非法拦截。
- 数量为负拦截。
- 订单关联缺失拦截。

## 5. SLA专项用例
1. 评审时效：1个工作日内完成。  
2. MRP时效：常规2个工作日、加急1个工作日。  
3. 外围时效：包装灭菌3天内完成。  
4. 常规交付：45天目标监控与告警。  

## 6. 缺陷判级
- 阻断：核心流程不可用。
- 高：关键功能偏差影响业务执行。
- 中：功能可用但存在明显体验或数据问题。
- 低：文案/展示类问题。

## 7. 通过标准
1. 阻断和高优先缺陷清零。  
2. 核心场景通过率达标。  
3. 回写与重排链路稳定。  
4. 测试报告可复现且可审计。  
5. 性能与同步补偿指标达标（计算P95、同步成功率）。  
6. 班次口径、人机料约束、WIP追溯相关用例全部通过。
