## 三、LLM 执行增强文档（新增）

1. `09_LLM重构执行手册.md`：新 LLM 接手后的启动顺序、硬约束、交接模板。

3. `11_契约冻结与变更流程.md`：接口冻结范围、变更分级、跨团队契约变更流程。
4. `12_模块LLM提示词库.md`：可直接下发给其他 LLM 的通用/分模块提示词模板。

## 四、执行顺序建议

1. 先读 `05`（统一目标与规则）。
2. 再按 `06` 认领模块与接口边界。
3. 新 LLM 首次接手必须先读 `09` 与 `10`，明确“任务-文件-测试”映射。
4. 涉及接口调整前必须先读 `11`，走契约变更流程。
5. 给其他 LLM 分派任务时优先使用 `12` 中对应模块提示词。
6. 开发与合并严格遵循 `07`（每次重构后必须验证通过）。
7. 每周按 `08` 执行阶段集成与风险复盘。
